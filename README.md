# manual

to run locally, in main folder run: 
```
python3 -m venv .venv 
source .venv/bin/activate
pip install mkdocs mkdocs-material
```

you can then run the website (when the venv is active)
```
mkdocs serve
```
This will allow you to visit the website locally. 
run 'deactivate' to get out of the virtual python environment
